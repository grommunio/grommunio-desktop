grommunio Desktop
=================

grommunio Desktop is the cross-platform client designed to run grommunio Web on
your desktop without any specific browser requirements. With grommunio Desktop,
users enjoy multi-account functionality with full integration of grommunio Web
including its integrations such as grommunio Files, Chat and Meet.

At a glance
===========

* Provides all web-based groupware functions (emails, contacts, calendar, tasks
  and notes) on your desktop.
* Works with all extensions integrating grommunio Meet, Chat Archive and more.
* Compatible, running on Microsoft Windows 8.1+, macOS 10.14+ and Linux (native
  packages to follow).
* Easy to use, providing a polished interface with nearly no training required
  for users.
* Fast, with a snappy interface which reacts almost immediately to user
  interactions.
* Secure, with certifications through independent security research and
  validation.

Compatibility
=============

* Microsoft Windows 8.1, 10, 11
* macOS 10.14 (Mojave), 10.15 (Catalina), 11 (Big Sur), 12 (Monterey), 13
  (Ventura); Intel (full), M1 (full) & M2 (in QA)
* Linux (x64)

Getting started
===============

Installation
------------

Installation images are to surface with official release:

* Linux: DEB/RPM, AppImage to follow
* macOS: DMG
